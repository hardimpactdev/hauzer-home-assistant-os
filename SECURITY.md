# Security policy

Please report suspected vulnerabilities to `security@hauzer.app` before opening a public issue. Include the affected version and reproduction steps, but never include access tokens, Home Assistant credentials, household data, or utility payloads.

We will acknowledge reports and coordinate fixes and disclosure with the reporter. Supported versions are the latest published release of the Home Assistant app.

If an import token may have been exposed, regenerate it immediately in Hauzer and update the app configuration.
